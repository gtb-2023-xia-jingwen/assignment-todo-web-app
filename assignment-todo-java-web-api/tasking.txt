0. 初始化
    - 与数据库创建连接，创建task表

1. 创建task
    - 启动Http服务器
    - 接收并处理POST请求
        - 创建task成功，返回201和task body
        - 参数校验异常，返回400
        - 服务端异常返回500

2. 获取task列表
    - 接收并处理GET请求
        - 参数校验
            - 不包含complete参数，返回所有task
            - 包含completed参数，值不为 true/false返回所有task，否则只返回已完成或未完成的task
        - 成功返回状态码200
        - 服务器端异常返回状态500

3. 更新task
    - 接收并处理PUT请求
        - 参数校验，检验失败返回状态码400
        - 参数校验成功
                -查询id是否存在
                    -不存在返回404；
                    -id存在执行更新操作，失败返回500，成功返回200
